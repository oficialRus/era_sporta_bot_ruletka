# Что сделать сейчас: картинка + приложение

## 1. Картинка «Колесо Фортуны»

**Сделано в коде:** путь к фото теперь ищется так:
- сначала `PROMO_IMAGE_PATH` из .env (если задан);
- затем `assets/wheel_promo.png` относительно текущей папки;
- затем `/root/era_sporta_bot_ruletka/assets/wheel_promo.png`.

Файл уже лежит: `assets/wheel_promo.png`.

**Нужно от вас:** перезапустить бота из **корня проекта**, чтобы подхватился новый код и нашёлся файл:

```bash
cd /root/era_sporta_bot_ruletka
./stop.sh
./start.sh
```

Если бот запускается через systemd:

```bash
sudo systemctl restart erasporta-bot
```

После этого при «С возвращением!» должна уходить картинка.

---

## 2. Приложение (рулетка в боте)

Чтобы работало открытие приложения по кнопке:

1. **Должны быть запущены API и веб-сервер:**

```bash
cd /root/era_sporta_bot_ruletka
./start.sh
```

(Это поднимет api, bot, web.)

2. **Nginx должен проксировать домен бота:**

```bash
sudo systemctl reload nginx
```

Если конфиг для бота ещё не подключали:

```bash
sudo ln -sf /etc/nginx/sites-available/bot-wheel-era-sporta.conf /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

3. **SSL для домена (если ещё не делали):**

```bash
sudo certbot --nginx -d bot-wheel.era-sporta-apsheronsk.ru
```

4. **Проверка:**

- В браузере: `https://bot-wheel.era-sporta-apsheronsk.ru/api/prizes` — должен вернуться JSON.
- В боте: «Открыть приложение» — должна открыться рулетка.
